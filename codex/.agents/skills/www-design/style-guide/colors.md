## System Colors

Semantic color sets for alerts/status. Each has a Copy (text), Border/Line, and Background/Flag variant.

### Warning

Amber tones — compliance risks, expiring items, attention needed.

| Role | Swatch |
|------|--------|
| Warning Copy | Dark amber (text on warnings) |
| Warning Line | Medium amber (borders) |
| Warning BG | Light amber (background fill) |

### Error

Red/coral tones — validation failures, destructive actions, critical alerts.

| Role | Swatch |
|------|--------|
| Error Copy | Dark red (text on errors) |
| Error Border | Medium red (borders) |
| Error Flag | Light red (background fill) |

### Success

Green tones — confirmations, completed actions, positive status.

| Role | Swatch |
|------|--------|
| Success Copy | Dark green (text on success) |
| Success Border | Medium green (borders) |
| Success Flag | Light green (background fill) |

## Gradient Colors

The design system includes brand gradients (Sky-to-Ocean, Gold, Coral, Midnight-to-Sky). Use sparingly — gradients are for hero/marketing surfaces, not UI controls.
